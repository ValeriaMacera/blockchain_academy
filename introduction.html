Introduction

What is the blockchain?

The Blockchain is defined by many experts as a ledger, a distributed ledger of transactions (DLT). It is safe, immutable, publicly accessible, auditable, uncensurable and decentralized.
It doesn't exist only one blockchain but there are different protocols and with the same protocol it could be created different blockchains. At the moment there are about a thousand of them but the number could grow endlessly.
Each blockchain can have a different name and the important thing is that each one has a token that is used within it in order to send transactions. It isn't necessary to use a cryptocurrency or a coin.

When did it all start?

On january 3 of 2009 there was the first event on Bitcoin's blockchain. There was the second bailout for banks.
Bitcoin was the first experiment and the first working system to use a blockchain.To know more details click here.
Before Bitcoin, many predecessors tried to create something similar. For example Stuart Haber and W. Scott Stornetta came up with the concept of a chain of blocks protected cryptografically, another example is the Reusable Proof of Work (RPoW) of Hal Finney and then the Bitcoin's Whitepaper was published by the creator Satoshi Nakamoto.
One of the main reasons for which the antecedents of Bitcoin failed was the fact that were incapable of put together a really distributed system on the Internet.
